Introduction

CSS (Cascading Style Sheets) controls how HTML content looks. It defines colors, spacing, layout, and typography across the web.

In this documentation, you'll learn core CSS concepts with examples you can copy and run.

CSS can be applied inline, embedded in the document, or linked as an external stylesheet.

/* Link a stylesheet */
<link rel="stylesheet" href="styles.css">
  • Separation of concerns: HTML for structure, CSS for style
  • Cascade and specificity control which rules win
  • Responsive design adapts to different screens
Selectors

Selectors target elements to style. They can be simple or complex.

p { line-height: 1.7; }
.highlight { background: yellow; }
#hero { padding: 2rem; }
a[target="_blank"] { text-decoration: underline; }
a:hover { text-decoration: none; }
  • Element: h1, p, button
  • Class: .card, .btn
  • ID: #navbar
  • Pseudo-class: :hover, :focus
Box Model

Every element is a rectangle composed of content, padding, border, and margin.

*, *::before, *::after { box-sizing: border-box; }
.card {
  width: 300px;
  padding: 1rem;
  border: 1px solid #ddd;
  margin: 1rem auto;
}
  • Content box: width excludes padding/border
  • Border box: width includes padding/border
Flexbox

Flexbox is a layout system for aligning items horizontally or vertically.

.toolbar {
  display: flex;
  gap: 1rem;
  justify-content: space-between;
  align-items: center;
}
.item { flex: 1 1 200px; }
  • Control axis with flex-direction
  • Align with justify-content
  • Wrap with flex-wrap
Grid

CSS Grid is a two-dimensional layout system.

.gallery {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(220px, 1fr));
  gap: 1rem;
}
.gallery__item--wide { grid-column: span 2; }
Typography

Good typography improves readability and hierarchy.

html { font-size: 16px; }
h1 { font-size: 2rem; }
p  { font-size: 1rem; line-height: 1.7; }
Color and Backgrounds

Colors set mood and accessibility.

.hero {
  background:
    linear-gradient(to bottom, rgba(0,0,0,.4), rgba(0,0,0,.4)),
    url('hero.jpg') center/cover no-repeat;
  color: white;
  padding: 4rem 2rem;
}
  • Meet WCAG contrast guidelines
  • Use system colors for theming
Responsive Design

Responsive design adapts layouts using media queries.

.card { padding: 1rem; }
@media (min-width: 768px) {
  .card { padding: 2rem; }
}
